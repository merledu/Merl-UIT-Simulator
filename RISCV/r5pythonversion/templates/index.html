About us

We are the students of Usman Institute of Technology, doing bachelors of computer science(4th year). We are doing this project as our final year project(FYP). Our website is about the simulation of RISC-V ISA(Instruction Set Architecture). The main Purpose of making this simulator is to get familiar with RISC-V ISA(Instruction Set Architecture).

RISC-V is an open-source hardware instruction set architecture (ISA) based on established reduced instruction set computer (RISC) principles. The project began in 2010 at the University of California, Berkeley, but many contributors are volunteers not affiliated with the university.
